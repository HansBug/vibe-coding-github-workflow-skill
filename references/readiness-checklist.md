# Readiness Checklist (WIP)

This reference will define reusable gates for deciding whether an issue or PR is ready for the next step.

Planned scope:

- issue planning readiness;
- empty PR contract readiness;
- reviewer pool discovery and selected-executor record;
- plan review readiness;
- implementation review readiness;
- CI, test, and Codecov readiness;
- ready-to-merge handoff while waiting for explicit maintainer instruction.

This file is intentionally a placeholder until the FS-B/FS-C phases expand the checklist into mechanical checks.
